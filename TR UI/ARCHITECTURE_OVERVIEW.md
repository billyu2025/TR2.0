## 📦 核心組件

### 前端 (Frontend)
- **技術棧**: Vue 3, JavaScript, HTML5, CSS3
- **頁面**: 登入、控制台、TR記錄管理、用戶管理
- **特點**: 響應式設計、單頁應用

### 後端 (Backend)
- **框架**: Flask 3.0.0
- **語言**: Python 3.x
- **API**: RESTful API (38 個端點)
- **服務器**: Gunicorn (Linux) / Waitress (Windows)

### 數據層 (Data Layer)
- **數據庫**: SQLite (data_3years.db)
- **緩存**: Redis (可選) / 內存緩存
- **文件**: 本地文件系統

## 🔐 安全機制

| 機制 | 實現 |
|------|------|
| **認證** | PBKDF2-HMAC-SHA256 密碼加密 |
| **授權** | 角色基礎訪問控制 (RBAC) |
| **限流** | flask-limiter (登入 5次/分鐘) |
| **會話** | JWT 風格令牌 (24小時) |

## ⚡ 性能優化

| 優化 | 技術 |
|------|------|
| **數據庫** | 連接池、WAL 模式、索引優化 |
| **緩存** | Redis/內存緩存、API 響應緩存 |
| **索引** | 文件索引系統、快速搜索 |
| **異步** | 下載任務異步處理 |

## 📊 數據流

```
用戶請求
    ↓
Nginx (反向代理)
    ↓
Flask 應用
    ├─ 認證檢查
    ├─ 限流檢查
    ├─ 緩存檢查
    ├─ 數據庫查詢
    └─ 響應返回
    ↓
用戶接收
```

## 🚀 部署架構

### Windows 生產環境
```
用戶 → Nginx → Waitress → Flask → SQLite
```

### Linux 生產環境
```
用戶 → Nginx → Gunicorn → Flask → SQLite
```
