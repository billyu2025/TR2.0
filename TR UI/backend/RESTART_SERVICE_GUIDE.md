# 重启服务指南（无需输入命令）

## 方法一：使用批处理脚本（最简单）✅

**直接双击运行：**

1. 找到文件：`C:\TR-master\TR UI\backend\restart_service.bat`
2. **双击运行**
3. 等待完成（会自动停止和启动服务）

---

## 方法二：使用 Windows 服务管理器（图形界面）

### 步骤：

1. **打开服务管理器**
   - 按 `Win + R`
   - 输入：`services.msc`
   - 按 Enter

2. **找到服务**
   - 在列表中找到 "TR Report System Backend"
   - 或按 `T` 键快速定位

3. **重启服务**
   - **方法 A**：右键点击服务 → 选择 "重新启动"
   - **方法 B**：
     - 右键点击服务 → 选择 "停止"
     - 等待服务停止
     - 右键点击服务 → 选择 "启动"

---

## 方法三：使用任务管理器

### 步骤：

1. **打开任务管理器**
   - 按 `Ctrl + Shift + Esc`
   - 或按 `Ctrl + Alt + Delete` → 选择 "任务管理器"

2. **切换到服务标签**
   - 点击 "服务" 标签
   - 或点击 "详细信息" → "服务" 标签

3. **找到服务**
   - 查找 "TR-Backend" 或 "TR Report System Backend"

4. **重启服务**
   - 右键点击服务 → 选择 "重新启动"
   - 或先 "停止" 再 "启动"

---

## 方法四：创建桌面快捷方式

### 创建快捷方式：

1. **找到脚本文件**
   - `C:\TR-master\TR UI\backend\restart_service.bat`

2. **创建快捷方式**
   - 右键点击 `restart_service.bat`
   - 选择 "发送到" → "桌面快捷方式"

3. **使用快捷方式**
   - 以后只需双击桌面上的快捷方式即可重启服务

### 可选：修改快捷方式图标

1. 右键点击快捷方式 → "属性"
2. 点击 "更改图标"
3. 选择你喜欢的图标

---

## 验证服务已重启

重启后，可以通过以下方式验证：

### 方法一：查看服务状态

1. 打开服务管理器（`services.msc`）
2. 查看 "TR Report System Backend" 的状态
3. 应该显示 "正在运行"

### 方法二：检查端口

1. 打开命令提示符（`cmd`）
2. 输入：`netstat -ano | findstr ":5000"`
3. 应该看到端口 5000 正在监听

### 方法三：访问系统

1. 打开浏览器
2. 访问：http://localhost:8000
3. 如果能正常访问，说明服务运行正常

## 故障排除

### 如果批处理脚本无法运行

**可能原因：**
- 需要管理员权限

**解决方法：**
1. 右键点击 `restart_service.bat`
2. 选择 "以管理员身份运行"

### 如果服务无法启动

**检查步骤：**
1. 查看错误日志：`C:\TR-master\TR UI\backend\logs\nssm_error.log`
2. 检查服务配置是否正确
3. 确保端口 5000 未被占用

---

**最简单的方法：双击 `restart_service.bat` 文件即可！**
