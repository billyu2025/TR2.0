## 📝 注意事项

1. **表名大小写**: PostgreSQL 中表名和列名需要使用引号包裹以保持大小写
2. **占位符**: 使用 `db_placeholders()` 函数而不是硬编码 `?` 或 `%s`
3. **行格式**: PostgreSQL 返回 `dict_row`，SQLite 返回 `tuple`，需要兼容处理
4. **函数差异**: 某些 SQL 函数在两种数据库中不同（如 `GROUP_CONCAT` vs `string_agg`）
