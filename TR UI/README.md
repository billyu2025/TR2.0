# TR Report System - 用户界面

## 项目说明

这是TR Report System的前端用户界面，使用Vue 3开发。

## 文件结构

```
TR UI/
├── index.html          # 系统入口页面
├── login.html          # 登录页面
├── dashboard.html      # 主控制台页面
├── styles/
│   ├── login.css      # 登录页面样式
│   └── dashboard.css  # 主控制台样式
├── scripts/
│   ├── login.js       # 登录页面逻辑
│   └── dashboard.js   # 主控制台逻辑
└── README.md          # 项目说明文档
```

## 使用方法

### 快速开始

**方式一：从入口页面开始**
1. 在浏览器中打开 `index.html`
2. 等待自动跳转或点击链接跳转到登录页面

**方式二：直接打开登录页面**
1. 在浏览器中打开 `login.html`

**登录流程：**
1. 选择用户类型（超级管理员或普通用户）
2. 输入账号名称和密码
3. 点击"登录"按钮
4. 登录成功后将自动跳转到主控制台
