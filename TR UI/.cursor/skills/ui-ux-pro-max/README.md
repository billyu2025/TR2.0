# UI UX Pro Max Skill for Cursor

This skill provides design intelligence for building professional UI/UX across multiple platforms.

## Usage

In Cursor, you can invoke this skill by typing:

```
/ui-ux-pro-max [your design request]
```

## Example Prompts

- "优化 TR记录管理页面的UI设计，使其更现代化和专业"
- "为TR Report System创建一个完整的设计系统"
- "改进表格的样式和交互体验"

## Features

- Automatic design system generation
- Smart style recommendations
- Best practices validation
- Multi-platform support (HTML, React, Vue, iOS, Android, etc.)
