## 系統概述

**TR Report System** 是一個用於管理測試報告（Test Report）的企業級 Web 應用系統，主要功能包括：

- 📊 TR 記錄查詢和管理
- 👥 用戶管理和權限控制
- 📄 PDF 報告生成和下載
- 🔍 文件索引和搜索
- 📦 批量下載管理
- 📈 數據統計和分析

### 系統特點

- ✅ **前後端分離**：前端使用 Vue 3，後端使用 Flask
- ✅ **RESTful API**：標準化的 API 接口設計
- ✅ **安全性**：密碼加密、會話管理、請求限流
- ✅ **高性能**：數據庫連接池、緩存系統、索引優化
- ✅ **可擴展**：模塊化設計，易於擴展

## 數據流

### 1. 用戶登入流程

```
用戶輸入憑證
    ↓
前端發送 POST /api/auth/login
    ↓
後端驗證用戶名和密碼
    ↓
生成會話令牌（Token）
    ↓
返回令牌和用戶信息
    ↓
前端存儲令牌
    ↓
後續請求攜帶令牌
```

### 2. TR 記錄查詢流程

```
用戶輸入查詢條件
    ↓
前端發送 GET /api/tr-records
    ↓
後端檢查緩存
    ├─ 有緩存 → 返回緩存結果
    └─ 無緩存 → 查詢數據庫
            ↓
        更新緩存
            ↓
        返回結果
    ↓
前端渲染數據
```

### 3. PDF 生成流程

```
用戶請求生成 PDF
    ↓
前端發送 POST /api/pdf/generate
    ↓
後端驗證權限
    ↓
查詢訂單數據
    ↓
渲染 PDF 模板
    ↓
生成 PDF 文件
    ↓
返回文件下載鏈接
    ↓
前端下載 PDF
```

### 4. 批量下載流程

```
用戶選擇多個訂單
    ↓
前端發送 POST /api/download/create-task
    ↓
後端創建下載任務
    ↓
異步處理下載任務
    ├─ 查找文件
    ├─ 打包 ZIP
    └─ 更新任務狀態
    ↓
前端輪詢任務狀態
    ↓
任務完成後下載 ZIP
```

## 安全機制

### 1. 認證和授權

**密碼安全：**
- ✅ PBKDF2-HMAC-SHA256 加密
- ✅ 120,000 次迭代
- ✅ 隨機鹽值（16 字節）
- ✅ 密碼過期機制（180 天）

**會話管理：**
- ✅ JWT 風格令牌
- ✅ 24 小時過期時間
- ✅ 令牌驗證中間件
- ✅ 登出時清除會話

**權限控制：**
- ✅ 角色基礎訪問控制（RBAC）
- ✅ 三種角色：`admin`, `manager`, `user`
- ✅ Job No 級別權限控制
- ✅ 端點級別權限驗證

### 2. 請求限流

**限流規則：**
- ✅ 登入：每分鐘 5 次
- ✅ 密碼重置：每小時 3 次
- ✅ 用戶註冊：每小時 10 次
- ✅ 全局默認：每天 200 次，每小時 50 次

**限流實現：**
- ✅ 基於 IP 地址限流
- ✅ 使用 `flask-limiter`
- ✅ 內存存儲（可升級到 Redis）

### 3. 數據安全

**SQL 注入防護：**
- ✅ 參數化查詢
- ✅ 輸入驗證和清理

**XSS 防護：**
- ✅ 輸出轉義
- ✅ Content-Type 設置

**CORS 配置：**
- ✅ 允許的來源配置
- ✅ 憑證支持

### 4. 錯誤處理

**錯誤響應：**
- ✅ 統一錯誤格式
- ✅ 不洩露敏感信息（生產環境）

```
開發者機器
    ↓
Flask 開發服務器 (localhost:5000)
    ↓
直接訪問前端文件 (file:// 或 localhost)
```

### 生產環境（Windows）

```
用戶瀏覽器
    ↓
Nginx (反向代理)
    ├─ 靜態文件服務
    └─ API 請求轉發
    ↓
Waitress (WSGI 服務器)
    ├─ 多線程處理
    └─ 生產環境配置
    ↓
Flask 應用
    ↓
SQLite 數據庫
```

### 生產環境（Linux）

```
用戶瀏覽器
    ↓
Nginx (反向代理)
    ├─ 靜態文件服務
    └─ API 請求轉發
    ↓
Gunicorn (WSGI 服務器)
    ├─ 多進程處理
    └─ 生產環境配置
    ↓
Flask 應用
    ↓
SQLite 數據庫
```

   - 訂單報告生成
   - 橫向 PDF 支持
   - PDF 下載

4. **文件下載**
   - 批量下載
   - 異步任務處理
   - 進度追蹤

   - 數據庫連接池
   - 緩存系統
